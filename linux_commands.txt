> ls -> para ver o que há no diretorio e "-la" para ver pastas com ponto (.exemplo)
> cd -> para mudar de diretorio onde estou
> which python3 -> para ver a versão e qual Python estou usando, se é do proprio sistema ou ambiente virtual
> python -m venv .venv -> criar um ambiente virtualizado
> source .venv/bin/activate ou bin/activate -> para ativar o ambiente virtual
> pwd -> diretorio onde estou


> git status -> para ver como está o projeto
> git add . -> adiciona todas as modificaçoes
> git commit -m "mensagem" -> truncam as mudancas
> git push origin main -> leva todas as mudacas para diretorio do GitHub

> rm -rf pasta -> remove a pasta(ex: pasta)
> touch file -> cria um arquivo
> nano file -> entra no arquivo e pode ser escrito/gravado
> mv file_or_directory_old file_or_directory_new -> este comando é usado tanto para pasta quanto para arquivo e serve
para renomear.
> pip freeze > requirements.txt -> bibliotecas não-padrão instaladas no projeto
> mkdir <pasta> -> cria uma pasta
> && -> operador para incluir outras instruções/comandos, ex: > touch file.txt && touch file2.txt -> cria dois arquivos de texto
> whoami -> mostra o usuário da sessão
> <tab> -> para completar nomes de arquivo, pastas...
> <right click> -> util se voce copia algo fora do terminal e quer colar(<right click>)
